## Сложение матриц
Матрицы должны быть одинакового размера, складываются поэлементно.
## Вычетание
Матрицы должны быть одинакового размера, вычитаются поэлементно.
## Умножение на число
Каждый элемент умножается на это число

## Уножение матриц
Матрицы можно перемножать, если кол-во столбцов первой матрицы равно кол-ву строк во второй матрице.
A = [
    [1, 2],
    [3, 4]
]

B = [
    [5, 6, 7],
    [8, 9, 10]
]

A x B = [
     [1*5+2*8, 1*6+2*9, 1*7+2*10],
     [3*5+4*8, 3*6+4*9, 3*7+4*10],
]
## Нахождение обратной матрицы
Обратная матрица — это матрица, которая при умножении на исходную матрицу в результате дает единичную матрицу (с единицами на диагонали и нулями в остальных ячейках).
